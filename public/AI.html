AI-Based Communication Skill Evaluation Web App

This project is a web-based communication skill evaluator that allows users to record their speech, transcribe it in real time, and receive an AI-generated score out of 100. It evaluates grammar, depth, and relevance of the spoken content. Scores are generated using SHA-256 to ensure consistency and fairness. The application features a modern UI with a video background, and stores top scores in a public leaderboard using local storage. It's built using HTML, CSS, and JavaScript with the Web Speech API.
