OLA ride bookings


This SQL project analyzes Ola ride bookings to derive insights into ride patterns, cancellations, and customer behavior. The dataset includes booking details, customer and driver interactions, payment methods, and ride distances. The SQL queries perform key operations such as filtering successful bookings, calculating average ride distances per vehicle type, and identifying cancellation reasons. The project aims to optimize ride efficiency, improve customer experience, and analyze cancellation trends. Insights gained will help refine Ola’s operational strategies, ensuring better service delivery and increased customer satisfaction.

Netflix Content Analysis and Insights Dashboard


This project analyzes Netflix’s content database to extract valuable insights using SQL and data visualization. It explores trends in movies and TV shows by type, release year, country, and director, and identifies popular genres, top actors, and the most common ratings. Key features include filtering content by year, director, or genre, tracking Indian content trends, and classifying shows based on themes like violence. The project supports data-driven storytelling through interactive visualizations and queries.
